# Your Italian Home — статический сайт

Простой одностраничный сайт для консультанта по недвижимости Людмилы (русский язык).

Структура:

- `index.html` — основной файл
- `style.css` — стили
- `script.js` — простая логика меню/формы
- `assets/hero.jpg` — замените на ваше фоновое изображение
- `assets/docs/` — добавьте реальные изображения дипломов и сертификатов

Запуск локально:

Для простого просмотра откройте `index.html` в браузере.
Для локального сервера (рекомендуется):

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

Замените файлы в `assets/` на реальные изображения документов и геро-фото.
